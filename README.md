# vue-components

This vue components consist of some important vue components that are custom built and is useful for any vue project. It features Vue 3.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
